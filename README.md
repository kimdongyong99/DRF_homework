## 프로젝트 구성 및 기능

프로젝트는 accounts, products 2개의 앱으로 구성되어있습니다.

products(게시기능)

-게시글 작성
-게시글 삭제
-게시글 수정
-상품목록 조회

### accounts

accounts(회원기능)

-회원 가입
-로그인
-프로필 조회
